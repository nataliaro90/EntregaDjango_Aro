# Entrega: Mi Primera Página Web en Django
**Alumno/a:** ARO NATALI

## Descripción
Este proyecto es un Blog de Viajes desarrollado bajo el patrón MVT de Django.

## Instrucciones de prueba
1. **Acceso al sitio:** Abrir `http://127.0.0.1:8000/blog/`
2. **Carga de datos:** - Usar el menú "Autor", "Post" o "Categoría" para insertar nuevos registros.
   - El sistema guardará los datos en la base de datos SQLite3.
3. **Búsqueda:** - Ir a la sección "Buscar".
   - Ingresar una palabra que coincida con el **Título** de un Post guardado para ver los resultados.
4. **Admin:** Acceder a `/admin` con el superusuario creado para gestionar los datos manualmente.
